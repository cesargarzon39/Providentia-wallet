# MVP 1: Autogestión de Salud
Basado en el pilar de "Health Wallet" de Elena.

## Descripción
Herramienta para que el paciente sea el dueño y gestor de sus propios datos de salud.

## Características Clave
- Contenedor de tokens firmados (RDA - Resumen Digital de Atención).
- Interoperabilidad mediante HL7 FHIR R4.
- Control de acceso: el paciente decide quién ve su historia.
